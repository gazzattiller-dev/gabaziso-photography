# Gabaziso Alfred Mtupa - Photography Portfolio

## Overview
A clean and responsive photography portfolio website built with HTML and CSS.

## Structure
- `index.html` — main website file
- `style.css` — styling file
- `images/` — folder for all portfolio images

## Add Your Images
Place your photos in the `images` folder using these names:
- hero.jpg
- portrait1.jpg – portrait4.jpg
- lifestyle1.jpg – lifestyle4.jpg
- event1.jpg – event4.jpg
- logo-placeholder.jpg

## Hosting Instructions
### Option 1: GitHub Pages
1. Create a GitHub account.
2. Upload this folder to a new repository.
3. Go to Settings → Pages → Select "main" branch → Save.
4. Visit your live link (e.g., `https://yourusername.github.io/portfolio`).

### Option 2: Netlify
1. Go to [netlify.com](https://www.netlify.com/).
2. Drag and drop the folder onto the dashboard.
3. Your website will be live instantly!

### Option 3: Any Hosting Service
Upload to any host (InfinityFree, Hostinger, etc.).
